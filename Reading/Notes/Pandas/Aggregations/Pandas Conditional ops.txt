############ Conditional transformation ################3

1> np.where use

flights_df['distance_km'] = np.where(
    flights_df['distance_unit'] == 'miles',
    flights_df['distance'] * 1.60934,
    flights_df['distance']
)

The function takes three arguments:

Condition: A boolean expression that evaluates to True or False for each row (distance_unit == 'miles')
Value if True: What to assign when the condition is True (multiply distance by 1.60934)
Value if False: What to assign when the condition is False (keep distance unchanged)


2> Multiple Conditionals
conditions = [flights_df['delay'] <= -5,
              flights_df['delay'] < 5,
              flights_df['delay'] >= 5]

outcomes = ['Early', 'On time', 'Late']

flights_df['delay_status'] = np.select(conditions, outcomes, default = 'Unknown)

The structure has three key components:

conditions: A list of boolean expressions—one for each category we want to create
outcomes: A list of values corresponding to each condition (must have the same number of elements)
default: The value to use when no condition matches (here we use 'Unknown' as a safety net)
np.select() evaluates conditions in order for each row. The first condition that's True determines the outcome, and the remaining conditions are skipped.

4> Using .apply()

airline_summary_df = (
    arrived_flights_df
    .groupby('airline')
    .apply(lambda x: pd.Series({
        'avg_delay': x['delay'].mean(),
        'avg_delay_delayed': x.loc[x['delay'] > 5, 'delay'].mean(),
        'total_passengers': x['passengers'].sum(),
        'passengers_delayed': x.loc[x['delay'] > 5, 'passengers'].sum()
    }), include_groups=False)
)

Here's how the pattern works:

apply(): gives us the full grouped DataFrame (not just one column)
lambda x: receives that grouped DataFrame
x.loc[x['delay'] > 5, 'passengers']: filters rows based on delay condition, then selects the passengers column
.sum(): aggregates the selected column
pd.Series({}): creates output with named columns
include_groups=False: prevents the grouping column from being passed into the lambda—without it, pandas includes the grouping column which can cause warnings
